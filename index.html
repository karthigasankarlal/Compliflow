Supabase setup guide
Follow these steps once. Takes about 45 minutes. After this, all 20+ users share live data and log in with email + password.
1
Create a free Supabase account

Go to supabase.com → click "Start your project" → sign up with Google or email. It's completely free for teams your size (up to 50,000 rows).

2
Create a new project

Click "New project" → give it a name like compliflow → choose a region closest to you (e.g. US East) → set a database password → click "Create new project". Wait 1–2 minutes for it to spin up.

3
Run this SQL to create your tables

In your Supabase dashboard → click "SQL Editor" in the left sidebar → paste the SQL below → click "Run".

-- Team members (mirrors your Settings > Team members) CREATE TABLE team_members ( id uuid PRIMARY KEY DEFAULT gen_random_uuid(), name text NOT NULL, email text UNIQUE NOT NULL, color text DEFAULT '#1E3A5F', role text DEFAULT 'preparer', created_at timestamptz DEFAULT now() ); -- Tax tasks CREATE TABLE tax_tasks ( id uuid PRIMARY KEY DEFAULT gen_random_uuid(), client text NOT NULL, form text, entity text, year text DEFAULT '2024', stage text DEFAULT 'Not started', preparer text, reviewer text, due_date date, internal_date date, priority text DEFAULT 'Medium', drive_link text, notes text, prep_hrs numeric DEFAULT 0, rev_hrs numeric DEFAULT 0, created_at timestamptz DEFAULT now(), updated_at timestamptz DEFAULT now() ); -- R&D tasks CREATE TABLE rd_tasks ( id uuid PRIMARY KEY DEFAULT gen_random_uuid(), client text NOT NULL, type text, entity text, year text DEFAULT '2024', stage text DEFAULT 'Scoping', preparer text, reviewer text, due_date date, internal_date date, priority text DEFAULT 'Medium', drive_link text, notes text, prep_hrs numeric DEFAULT 0, rev_hrs numeric DEFAULT 0, created_at timestamptz DEFAULT now(), updated_at timestamptz DEFAULT now() ); -- Enable realtime updates (so all users see changes instantly) ALTER TABLE tax_tasks REPLICA IDENTITY FULL; ALTER TABLE rd_tasks REPLICA IDENTITY FULL; ALTER PUBLICATION supabase_realtime ADD TABLE tax_tasks; ALTER PUBLICATION supabase_realtime ADD TABLE rd_tasks;
4
Get your API keys

In the Supabase dashboard → click "Project Settings" (gear icon) → click "API" → copy two things: Project URL (looks like https://xxxx.supabase.co) and anon public key (a long string starting with eyJ...).

5
Paste your keys into this app

Replace the two placeholders at the very top of the JavaScript section of compliflow-supabase.html with your real keys:

const SUPABASE_URL = 'https://YOUR_PROJECT.supabase.co'; const SUPABASE_ANON_KEY = 'YOUR_ANON_KEY_HERE';
6
Add your team members in Supabase Auth

In Supabase dashboard → click "Authentication" → click "Users" → click "Invite user" → enter each team member's email. They'll receive an email to set their password. Alternatively use "Add user" to set a temporary password yourself and share it with them.

7
Host the file so everyone can open it

Option A — Netlify Drop (easiest, free): go to app.netlify.com/drop → drag and drop your compliflow-supabase.html file → Netlify gives you a URL like https://your-name.netlify.app — share that URL with your team. Takes 2 minutes.

Option B — save it to a shared Google Drive folder. Each team member downloads it and opens it in Chrome. (Works but everyone needs to re-download when you update it.)

8
You're live!

Everyone opens the URL, logs in with their email + password, and sees the same shared data. Changes made by one person are visible to all others within a few seconds thanks to Supabase Realtime.
